# BABA CONCEPT - Site Next.js

Site vitrine Next.js (App Router) pour BABA CONCEPT.

## Prérequis

- Node.js 20+
- npm

## Lancer en local

1. Installer les dépendances:
   `npm install`
2. Démarrer le serveur de développement:
   `npm run dev`
3. Ouvrir [http://localhost:3000](http://localhost:3000)

## Build de production

`npm run build`

## Assets

- Logo header: `/public/images/header-logo.jpg`
- Image header: `/public/images/header.png`
- Logo footer: `/public/images/footer-logo.jpg`
- Favicon: `/src/app/icon.png` et `/public/images/favicon-192.png`
- WhatsApp SVG officiel: `/public/images/whatsapp-official.svg`

## Déploiement Vercel

- Framework preset: `Next.js`
- Build command: `npm run build`
- Root directory: racine du dépôt
